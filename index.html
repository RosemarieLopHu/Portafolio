DataDashboard

El reto de este proyecto fue crear una interfaz donde las Training Managers (TMs) puedan ver y usar la data de progreso del Learning Management System (LMS) de las estudiantes, para ello se elaboró un data dashboard (tablero de visualización de datos).

Red Social: Healthy Food

Healthy Food es una red social cuyo objetivo es que los usuarios recomienden recetas, tiendas, productos, restaurantes que cuenten con los productos y/o servicios de alimentación saludable. Además de compartir los beneficios de llevar una alimentación saludable.

Markdown Links

Markdown Links

El objetivo práctico de este proyecto es que aprendas cómo crear tu propia librería (o biblioteca - library) en JavaScript. Diseñar una librería es fundamental porque que te permite pensar en la interfaz (API) de los módulos y como serán usados por otros developers, se tuvo especial consideración en peculiaridades del lenguaje, convenciones y buenas prácticas.
